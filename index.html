    Image and Source Rules

    This template intentionally uses no scraped web photos. Production pages should use official facts, authorized visitor notes, licensed images, and visible attribution.

    • Official text is used for factual extraction, not copied as prose.
    • AI summaries must trace important claims back to source records.
    • Visitor notes require permission to publish and translate.
    • Images need ownership, open license, or written authorization.
